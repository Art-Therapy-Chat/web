# 챗쪽이 - HTP 심리 검사 해석 챗봇 🎨🧠

AI 기반 HTP(House-Tree-Person) 심리 검사 해석 서비스입니다. 
그림을 그리거나 업로드하면 AI가 심리 상태를 분석하고, 대화형 인터페이스로 추가 정보를 수집하여 개인화된 해석을 제공합니다.

## ✨ 주요 기능

### 1. 그림 입력
- **직접 그리기**: 브러시 크기와 색상을 조절하여 캔버스에 직접 그림 그리기
- **이미지 업로드**: 이미 그려진 그림을 업로드하여 검사 진행
- **3가지 주제**: 집(House), 나무(Tree), 사람(Person) 각각 독립적으로 작업

### 2. 단계별 해석
- **개별 해석**: 집, 나무, 사람 각각에 대한 개별 분석 제공
- **대화형 정보 수집**: AI가 5가지 질문으로 추가 정보 수집
  - 연령, 성별, 현재 상황
  - 최근 기분과 감정 상태
  - 그림의 구체적 요소 (색상 선택 이유, 세부 설명 등)
- **최종 종합 해석**: 모든 정보를 종합한 개인화된 심리 분석

### 3. 지속적인 상담
- 최종 해석 후에도 계속 대화 가능
- 해석에 대한 추가 질문 및 상담 진행

## 🖼️ 스크린샷

### 1단계: 그림 입력
- 3개 탭(집, 나무, 사람)으로 구성
- 브러시 크기 및 색상 선택 가능
- 지우기 및 이미지 업로드 기능

### 2단계: 해석 결과
- 개별 해석 (집/나무/사람)
- 대화형 추가 정보 수집 (5개 질문)
- 최종 종합 해석
- 지속적인 채팅 상담
